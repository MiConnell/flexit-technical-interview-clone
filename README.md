# dbt Business Intelligence

This project shows how to structure dbt to enable a business intelligence tool for reporting and dashboarding.

## Getting started

dbt client, run the following commands:

First run:
* `dbt deps`
* `dbt seed`

Schedule:
* `dbt run`
* `dbt source freshness`

Docs:
* `dbt docs generate`
* `dbt docs serve`
